# FilminfoTMDB
Filminfo - Movie Database app using TMDB API

How to run app:
1. Clone this repository
2. Open Filminfo.xcodeproj

Build using SwiftUI, MVVM design pattern, Alamofire, YoutubePlayerKit with TMDB Api
